msxr
====

random music generator using bu-atrk-c and hxmikmod

features
 - set some parameters
 - hit Generate!
 - this will build the song to soundbuffer
 - hit Play to listen
 - hit Export to get .wav
 - get .it file
 - seek throughout the song

GUI
 - ... atrk params
 - Genarate!
 - Export .wav
 - Save .it
 - Play/Stop
 - seekbar
